# Executar o Código

Para executar o código basta entrar na pasta raiz e realizar o seguinte comando:

```bash
node index.js N
```

Onde N é o tamanho do tabuleiro.

**OBS:** É preciso ter o NodeJS versão 12.13.0 instalada em seu computador
