

O Programa deve ser executado em uma maquina usando Ubuntu/Bash Ubuntu, com a versão do GCC 4.8.4, e MAKE 3.81.

Para criar o programa basta executar o comando make, ele ira gerar o arquivo tp2.

Executando: basta usar este comando:

./tp2 -v entrada.txt -m movimentos.txt -B/-V/-H

-B Executa no modo BackTrack.

-V Verifica se o arquivo de manobras tem manobras validas.

-H Executa a heurística criada.
